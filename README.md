# todo

A simple daily routine tracker built as a single HTML file.

## Features

- Morning and evening routines with timed subtasks.
- `Apply M` and other steps show their scheduled time ranges.
- Custom tasks can be added for any selected day (past, present, or future).
- Per-day task state is stored in browser `localStorage`.
- Timers are available for timed steps like minoxidil, oil wait, and whitening strips.

## Usage

1. Open `index.html` in a browser.
2. Select a day using the week toggle or calendar.
3. Toggle tasks to mark them done.
4. Add extra tasks in the `Add task` section for any day.
5. Custom tasks are saved automatically in the browser.
